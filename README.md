# L'Archipel Créatif — Frontend (V1)

Codebase React + TypeScript + Vite + Tailwind + Wouter.

## Lancer en local
```bash
npm install
npm run dev
```

## Notes
- Le formulaire d'inscription est multi-étapes (3 étapes) avec logique de formules.
- Les inscriptions sont sauvegardées en localStorage (V1).
- Phase 2 : brancher Directus (back-office) + emails + export Excel côté backend.
