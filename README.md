# archipelcrea
L’Archipel Créatif est une plateforme immersive dédiée à l’éveil artistique des enfants à travers cinq îles thématiques. Chaque île propose un parcours structuré mêlant dessin, peinture, volume et artisanat. Le projet repose sur React et Directus pour offrir une expérience moderne, évolutive et administrable via un back-office.
